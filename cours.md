La commande 
```
    git init 
```
permet  d'initialiser un depot GIT. Elle retourne :
```
Initialized empty Git repository in C:/Users/Mywin/BTS/.git/
```

Pour vérifier l'état de votre depot GIT utiliser la commande : 
```
git status
```
